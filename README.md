# Thorium Mixer Mod

A mod that adds the Thorium Mixer and Radiofluid.

## Features

- **Thorium Mixer:** A new building to process thorium.
- **Radiofluid:** A new fluid used in advanced processes.

## Installation

To install the Thorium Mixer Mod, follow these steps:

1.  Download the latest release from the [releases page](https://github.com/tiagocunhapereira-br-bootdev/CapStone/releases).
2.  Place the downloaded `.zip` file into your Mindustry mods folder.
    -   **Windows:** `C:\Users\YOUR_USERNAME\AppData\Roaming\Mindustry\mods\`
    -   **Linux:** `~/.local/share/Mindustry/mods/`
    -   **macOS:** `~/Library/Application Support/Mindustry/mods/`
3.  Launch Mindustry and enable the mod in the "Mods" menu.

## Compatibility

This mod requires Mindustry version `158` or higher.
